# VPS-Security-Check
A single-page, beginner-friendly guide to hardening a Linux server: system updates, custom SSH port, firewall (ufw), IP allowlist (hosts.allow/deny), Fail2ban, and MaxAuthTries. Click-to-copy commands, no jargon. Frost white, gold, and bubblegum pink dark theme. One HTML file, no build. Ready for GitHub Pages. By AlexRabbit.
